# Monitor de Integridade de Arquivos

Projeto focado em cibersegurança para demonstrar:
- entendimento de hash (SHA256)
- verificação de integridade de arquivos
- detecção de alterações não autorizadas

## Objetivo
Criar um script que gere hashes de arquivos e depois compare
para identificar se houve modificação.

## Tecnologias
- Bash
- SHA256
- Linux
